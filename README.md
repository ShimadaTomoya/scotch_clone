# scotch_clone
- https://github.com/ng3rdstmadgke/scotch
  - をRubyで写経したものです

# usage
- Rubyをインストールしてください

- 開始
```
git clone git@github.com:ShimadaTomoya/scotch_clone.git
cd scotch_clone
bundle install --path vendor/bundle
bundle exec ruby main.rb
```

- 再開
```
<ctrl + c>
bundle exec ruby main.rb --continue
```
